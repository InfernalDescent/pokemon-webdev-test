Basic information / Backstory

Hi there! Thanks for visiting my project. I’m currently learning coding, and this started as a personal side project to create something I have some interest in, and to practice my skills. It’s always evolving, so be sure to check back often!

How the site works

Here’s how the site works: each generation link leads to maps where you can click routes to see Pokémon and items available there. Maps show game progression, like, for example, the earliest you can get a certain Pokémon or TM/HM. I also plan to add a checklist feature to track your goals. For now, only Gen 1 is done, but more will come!
